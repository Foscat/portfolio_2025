CodeStream Studios LLC.

Front End Developer 2019-2024

Overview

During my tenure at CodeStream Studios LLC, I served as the sole front-end developer, entrusted with the monumental task of crafting our React.js application from its conceptual infancy to its full-fledged existence. With a meticulous blend of programming prowess and strategic planning, I orchestrated the transformation of abstract ideas into tangible features that formed the cornerstone of our platform. As the architect behind the entire front-end framework, I not only coded every aspect of the website but also navigated the challenges of a lean team and limited resources. Amidst these constraints, I assumed a multifaceted role, extending beyond programming to encompass responsibilities such as bug fixing, documentation creation, and user testing management. This comprehensive approach allowed me to not only bring the platform to life but also ensure its functionality and success.

In addition to my role as a developer, I also had the unique opportunity to serve as an instructor and active user of the platform. Utilizing the very system I helped create, I seamlessly integrated it into my teaching curriculum, leveraging its capabilities to educate and empower students. This firsthand experience as both developer and user provided invaluable insights, allowing me to refine and optimize the platform with a deep understanding of its practical applications. Through this dual perspective, I not only contributed to its development but also witnessed firsthand its impact and efficacy in real-world educational settings.

Key Features

Our platform was designed to provide a comprehensive virtual learning experience for middle and high school students, offering a range of features tailored to meet the demands of remote education during the COVID-19 pandemic and beyond. Here's a brief overview of the core functionalities:

  • Classroom System: Our intuitive classroom system allowed educators to create and manage virtual classrooms, facilitating seamless interaction and collaboration among students and teachers.
  • 3 Panel Code Editor: Built with simplicity and functionality in mind, our innovative code editor featured a three-panel interface, empowering students to write and debug code in real-time without the need for additional software installation.
  • Grading and Reports: Streamlining the assessment process, our platform offered robust grading functionalities, enabling educators to evaluate student submissions efficiently and generate comprehensive reports to track progress and performance.
  • Curriculum Builder: In response to the evolving needs of educators and organizations, we developed a powerful curriculum builder tool, empowering users to create and customize lesson plans and course materials tailored to their specific teaching objectives.

Classroom System

The Problem:

We need a way to allow users who are teachers for an organization to have a class with curriculum to present to students. When the student is in a classroom they can view lessons and resources that are part of the course. Both students and teachers will need a standard way to access the classroom.

The Solution:

We added Classrooms as a tab on the side navigation that all users can see. Once navigating to the page users would see a list of all the classrooms they are a part of both teacher and student. The classrooms where the user is a teacher display the student count of the classroom. Clicking on a classroom will navigate the user to the Classroom page where all the information about the course is displayed. Here users can view resources attached to the course, classroom or lessons as well as open a lesson as a project. If the user currently does not have a project that matches this lesson it will be cloned and saved as a project under the users name and gets a grade entry attached to it so the teacher cn grade the project later.

Virtual Classroom

On the classroom page, we display easy access to lesson materials, assignments, and supplementary resources. The inclusion of an introductory video offers a dynamic introduction to the course content, enhancing engagement and comprehension for students.

Roster

The roster displays a comprehensive list of enrolled students, allowing instructors to efficiently track attendance and monitor participation. Additionally, the option to view each student's projects provides educators with a holistic view of individual student performance, facilitating personalized feedback and support.

Web Projects

The left panel features clear instructions and a robust set of tools, empowering learners to navigate the coding process with ease. In the middle panel, students can access and edit their code in real-time, with convenient tabs for switching between HTML, CSS, and JS files, facilitating seamless multi-language development. On the right panel, a mini web browser displays the live output of the student's code in a custom i-frame, allowing for immediate feedback and debugging.

Python Projects

The left panel retains familiar tools and lesson information, ensuring consistency in the user experience. However, the middle panel adapts to the unique requirements of Python development, displaying a single file interface optimized for Python coding projects. Meanwhile, the right panel transforms into a dedicated terminal output section, providing students with essential feedback and execution results specific to Python programming.

Grading Projects

Teachers can seamlessly view a student's project, offer personalized feedback, and assign a grade, with the flexibility to choose between interchangeable pass/fail or numeric/letter grading styles. Students can conveniently access this feedback from their teachers directly within the platform, fostering a collaborative and transparent learning environment.

Classroom Report

The report displays each student's name alongside their respective grades for all projects, providing a clear overview of individual achievements. Additionally, the platform calculates each student's average grade and progress in the course, facilitating informed decision-making and targeted support. Moreover, the option to toggle between 'Numeric' and 'Letter' grade views allows for customizable assessment perspectives, ensuring flexibility and accuracy in grading practices.

Creating New Content

On the course builder users can seamlessly craft courses by adding lessons and resources, structuring content to suit the needs of their curriculum. Once a lesson is integrated into a course, users have the flexibility to attach resources directly to individual lessons, enhancing the accessibility and comprehensiveness of course materials.

Modular Structure

Users are presented with a convenient popup menu for adding lessons to course units within the CSOS platform. The menu displays a table of lessons available within the user's organization, showcasing the modular nature of lessons and resources. Users can easily select and add lessons from this centralized repository, promoting efficiency and flexibility in curriculum development. This feature underscores the platform's commitment to modularity, allowing lessons and resources to be utilized across multiple courses and units.

Curriculum Dashboard

All curriculum creation and management activities are centralized under the 'Active Organization,' streamlining administrative processes and promoting organizational cohesion. Users can easily access and manage all curriculum assets within this dedicated dashboard, facilitating effective curriculum planning and oversight.
